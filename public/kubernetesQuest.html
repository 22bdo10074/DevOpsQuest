Kubernetes (or K8s) is an open-source container orchestration platform that automates the deployment, scaling, and management of containerized applications.

  • Cluster: A set of nodes (machines) that run containerized applications.
  • Master Node (Control Plane): Manages the Kubernetes cluster. Components include:
    • API Server: Handles communication between users and the cluster.
    • etcd: A key-value store maintaining cluster state.
    • Scheduler: Decides which node runs which pod.
    • Controller Manager: Ensures the desired state of the cluster.
  • Worker Node: Runs application workloads using:
    • Kubelet: Agent that runs containers and reports node health.
    • Kube Proxy: Manages networking and routing.
    • Container Runtime: Runs containers (Docker, containerd, etc.).
  • Pod: Smallest deployable unit in Kubernetes ‚Äî one or more containers.
  • Deployment: Manages updates and scaling of Pods.
  • Service: Exposes Pods to the network.
  • ConfigMap & Secret: Store configuration and sensitive data.
